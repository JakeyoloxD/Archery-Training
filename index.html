üìã Quick Reference: Top 10 Exam Facts

  • 1. Click SAVE to save record changes - not clicking loses all changes
  • 2. Click the logo (top left) to return to homepage/default workspace
  • 3. Red asterisks (*) = Required fields
  • 4. Valid access rights: CREATE, READ, UPDATE, DELETE (NOT append)
  • 5. Everyone group = all users automatically belong
  • 6. Issues Management = Use Case seen in nearly every Solution Area
  • 7. Data Feeds = import INTO Archer; Data Publications = export OUT OF Archer
  • 8. Personal reports = only you; Global reports = can be shared
  • 9. Keyword search searches within Applications (not workspaces/reports)
  • 10. Calculated fields auto-populate based on other fields
